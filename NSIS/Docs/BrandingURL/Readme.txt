一个NSIS插件，可将品牌文本设置为可点击的超链接。
 --------------------------------------------------------------
将BrandingURL.dll放置在您的NSIS\Plugins文件夹中，
或者只需将Zip中的所有文件解压缩到NSIS\中即可。

请参见Examples\BrandingURL\ExampleMUI.nsi，以查看用法示例。


 ==============================================================
函数：

BrandingURL::Set /NOUNLOAD [color_r] [g] [b] [link_url]

将品牌文本设置为可点击的超链接。

[color_r] : 超链接颜色的红色部分。
[g] : 超链接颜色的绿色部分。
[b] : 超链接颜色的蓝色部分。
[link_url] : 默认网页浏览器中打开的网址。

注意：此函数应在.onGUIInit中调用。
/NOUNLOAD是必需的。
对于r g b，使用空的""值以使用默认链接颜色。

          -------------------------------------------

 BrandingURL::Unload

卸载插件DLL以进行删除。

注意：此函数应在.onGUIEnd中调用。

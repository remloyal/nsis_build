FreeArc for NSIS 该插件为 NSIS 脚本提供了基本的 FreeArc 文件存档提取功能。

如果你想使用此插件，你必须使用 FreeArc 来压缩你的存档。
http://freearc.org/ 

测试和安装 FreeArc for NSIS 插件

- 将 Plugin\FreeArc.dll 复制到 NSIS\Plugin 目录。
- 编译 Example\Example.nsi 并进行测试。


仅限程序员：
要编译插件源代码，您必须从 http://freearc.org/  下载 FreeArc 源代码。
遵循 FreeArc 的编译说明。
然后复制并替换我的源代码文件到 "C:\!\FreeArchiver\Unarc"。
